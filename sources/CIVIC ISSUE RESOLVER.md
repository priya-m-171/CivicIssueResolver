# 		**CIVIC ISSUE RESOLVER**



**USER**

* register/login
* allow permissions
* select category
* upload photo
* fill details
* permissions=allow?"submit complaint" else "allow permissions and submit complaint"
* generate a ticket
* receive notifications



**AUTHORITY**

* login
* allow permissions
* view complaints
* verify complaints
* assign to workers
* update status(Acknowledged → Work Started → Completed)
* close complaint
* view analytics



**EMPLOYEES**

* login
* allow permissions
* view assigned complaints
* accept
* visit issue location
* perform repair
* update status
* upload proof





**ADMIN**

* manage users
* monitor system health
* generate reports
* maintain security and updates
